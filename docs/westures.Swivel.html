    westures.Swivel(optionsopt)

    A Swivel is a single input rotating around a fixed point. The fixed point is determined by the input's location at its 'start' phase.

    Properties
    Name Type Attributes Default Description
    deadzoneRadius number <optional>
    		 10 The radius in pixels around the start point in which to do nothing.
    enableKey string <optional>
    		 One of 'altKey', 'ctrlKey', 'metaKey', or 'shiftKey'. If set, gesture will only be recognized while this key is down.
    pivotCenter Element <optional>
    		 If set, the swivel's pivot point will be set to the center of the given pivotCenter element. Otherwise, the pivot will be the location of the first contact point.
